## Usage Rules

- Each variant leads with verifiable facts, not adjectives
- None claim titles not held
- None imply a degree
- None use filler ("proven ability", "passionate about", "detail-oriented")
- Nexus WiFi is framed as support/triage, not engineering
- Live Pilot is referenced as a system built, not a product shipped (that's Impulse)
- All numbers are sourced: 150+ properties, 1,500+ hours, 62 functions, 4+ years
- Customize the last sentence per application when needed
